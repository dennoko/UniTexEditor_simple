# UniTexEditor_simple - 変更履歴

## [0.3.0] - 2025-11-06

### 新機能
- **シャープネス / ぼかし機能を追加**
  - Unsharp Mask方式のシャープネス
  - ガウシアンブラー
  - カーネルサイズ調整可能（3/5/7/9）
  - マスク対応

### 改善
- **メッシュ指定機能の強化**
  - GameObject指定時に自動でメッシュを抽出
  - MeshFilter対応
  - SkinnedMeshRenderer対応
  - メッシュ情報の表示改善（頂点数表示）

### 修正
- メッシュ指定時にUVアイランドブラーが動作しない問題を修正

## [0.2.3] - 2025-11-06

### 修正
- パラメータ調整時にプレビューが黒くなる問題を修正
  - Compute Shaderでマスクがnullの時のエラーを解決（ダミーテクスチャを設定）
  - `RenderTextureFormat.RGBA32` → `RenderTextureFormat.ARGB32` に修正
  - ColorCorrectionNode、BlendNode、ToneCurveNodeすべてで対応

### 改善
- Compute Shader呼び出しの安定性向上
- エラーメッセージの改善
- マスク機能を常時ONに変更（UIの簡素化）

## [0.2.2] - 2025-11-06

### 修正
- プレビュー表示が真っ黒になる問題を修正
  - ノードが追加されていない場合にソーステクスチャを直接表示するように改善
  - GUI.DrawTextureの描画タイミングを最適化
  - 初回プレビュー生成の確実性を向上
  
### 改善
- デバッグログの追加（プレビュー生成状況を確認可能）
- メモリ管理の改善（OnDisable時にプレビューテクスチャをクリーンアップ）
- ノードなし時のフォールバック処理を追加

## [0.2.1] - 2025-11-05

### 修正
- プレビュー表示領域が消える問題を修正（GUILayoutの実装を簡素化）
- プレビュー描画タイミングの最適化（Event.current.type == EventType.Repaint）

### 改善
- プレビュー計算解像度を512x512に制限（パフォーマンス向上）
- 保存時はフル解像度で処理（品質維持）
- GUI.DrawTextureを使用してより安定した描画

## [0.2.0] - 2025-11-05

### 追加
- トーンカーブによる色調整機能（RGB/R/G/B 個別調整）
- プレビュー表示の改善（正方形維持、最大化表示）
- チェッカーボード背景（透明度確認用）

### 修正
- プレビュー表示が真っ暗になる問題を修正（RenderTexture解放タイミングの改善）
- プレビューのメモリリーク対策
- UI再描画の最適化

### 改善
- プレビューが表示領域に可能な限り大きく表示されるように改善
- アスペクト比を維持しつつ中央配置
- プレビュー情報（解像度）の表示

## [0.1.0] - 2025-11-05

### 追加
- 初期リリース
- 色調補正機能（Hue/Saturation/Brightness/Gamma）
- HDR色合成（加算、乗算、オーバーレイ等）
- テクスチャ合成（複数ブレンドモード対応）
- マスク機能（適用範囲制御、強度調整）
- UVアイランドブラー（境界を越えないブラー処理）
- GPU Compute Shader 実装
- エディタウィンドウUI
- 非破壊ワークフロー（新規保存/上書きオプション）
- 基本的なテストツール

### 既知の問題
- 一部のプラットフォームでCompute Shaderが動作しない可能性があります
- 大きなメッシュ（頂点数が多い）のUVアイランド解析に時間がかかる場合があります
- マスク反転機能は今後実装予定

### 今後の予定
- プリセット保存/読み込み機能
- レイヤーシステムの強化
- CPU フォールバック実装
- パフォーマンス最適化
- 追加のブレンドモード
- ヒストリー機能（Undo/Redo）
