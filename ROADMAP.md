# UniTexEditor_simple - 機能拡張ロードマップ

## コンセプト
**画像全体に対する調整機能**に特化し、加筆・ペイント等のローカル編集は行わず、
フォトレタッチやテクスチャ最終調整で頻繁に使用される機能をカバーする。

---

## 🎯 優先度：高（すぐに追加したい機能）

### 1. **シャープネス / ぼかし**
- **目的**: エッジの強調、ノイズ低減、ソフトフォーカス効果
- **実装方法**: 
  - Unsharp Mask（アンシャープマスク）
  - ガウシアンブラー
  - ComputeShaderで畳み込みフィルタ実装
- **パラメータ**: 
  - シャープネス強度（0.0 ~ 2.0）
  - ぼかし半径（0 ~ 20ピクセル）

### 2. **ノイズ除去 / グレイン追加**
- **目的**: テクスチャのクリーンアップ、フィルム調グレイン効果
- **実装方法**:
  - メディアンフィルタ（ノイズ除去）
  - パーリンノイズ / ホワイトノイズ合成
- **パラメータ**:
  - ノイズ除去強度
  - グレイン量・サイズ・モノクロ/カラー

### 3. **レベル補正**
- **目的**: ヒストグラム調整による明暗コントラスト制御
- **実装方法**:
  - 入力レベル（黒点・白点・中間点）
  - 出力レベル（出力の黒・白クリッピング）
- **パラメータ**:
  - Input Min/Mid/Max
  - Output Min/Max
  - チャンネルごとの調整（RGB/R/G/B/A）

### 4. **ホワイトバランス**
- **目的**: 色温度の補正、照明環境の違いを調整
- **実装方法**:
  - 色温度スライダー（2000K ~ 10000K）
  - 色偏差補正（グリーン/マゼンタバランス）
- **パラメータ**:
  - Temperature（色温度）
  - Tint（色偏差）

---

## 🔧 優先度：中（有用だが実装が複雑）

### 5. **自動レベル調整 / 自動コントラスト**
- **目的**: ワンクリックで最適化
- **実装方法**:
  - ヒストグラム分析でMinMax自動検出
  - ガンマ自動調整

### 6. **カラーマッチング / カラーグレーディングLUT**
- **目的**: 特定の色調に統一、シネマティック効果
- **実装方法**:
  - 3D LUT（Look-Up Table）テクスチャ適用
  - プリセット（Cinematic / Vintage / Cool / Warm等）
- **パラメータ**:
  - LUTテクスチャスロット
  - 適用強度

### 7. **チャンネルミキサー**
- **目的**: 各色チャンネル間の影響をコントロール
- **実装方法**:
  - R/G/B出力チャンネルごとに入力R/G/Bの配合を調整
- **パラメータ**:
  - Red Output: R, G, B影響度
  - Green Output: R, G, B影響度
  - Blue Output: R, G, B影響度

### 8. **グラデーションマップ**
- **目的**: モノクロ化 + カラーグラデーション適用
- **実装方法**:
  - グレースケール化後、輝度値をGradientテクスチャにマップ
- **パラメータ**:
  - グラデーションテクスチャスロット
  - ブレンド強度

---

## 🌟 優先度：低（特殊用途・エクストラ機能）

### 9. **ビネット効果**
- **目的**: 周辺減光でフォーカス効果
- **実装方法**:
  - 中心からの距離でアルファブレンド
- **パラメータ**:
  - 強度・範囲・形状（円形/楕円）

### 10. **クロマティックアバレーション（色収差）**
- **目的**: レンズ効果、アート表現
- **実装方法**:
  - RGBチャンネルを微妙にオフセット
- **パラメータ**:
  - 強度・中心位置

### 11. **エッジ検出 / アウトライン生成**
- **目的**: トゥーンシェーディング用、デバッグ用
- **実装方法**:
  - Sobelフィルタ、Prewittフィルタ
- **パラメータ**:
  - 感度・太さ・色

### 12. **タイリングチェック / シームレステクスチャ対応**
- **目的**: タイリング継ぎ目の確認
- **実装方法**:
  - プレビューを2x2タイル表示モード
  - エッジブレンド機能

## 次回実装候補（v0.3.0）

以下の3つを次バージョンで追加推奨:
1. **シャープネス/ぼかし** - ComputeShaderでUnsharp Mask実装
2. **レベル補正** - Input/Output Min/Mid/Max制御
3. **ホワイトバランス** - 色温度・Tint調整

これらは全て**画像全体への調整**で、現在のノードシステムに容易に統合可能。
