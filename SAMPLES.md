# UniTexEditor_simple - サンプルとチュートリアル

## 基本的な使い方

### 1. エディタウィンドウを開く
Unity メニューから `Window > UniTex Editor` を選択します。

### 2. ソーステクスチャを指定
エディタウィンドウの「ソーステクスチャ」フィールドに編集したいテクスチャをドラッグ＆ドロップします。

### 3. パラメータを調整
- **色相シフト**: -180〜180の範囲で色相を回転
- **彩度**: 0〜2の範囲で色の鮮やかさを調整
- **明度**: 0〜2の範囲で明るさを調整
- **ガンマ**: 0.1〜3の範囲でガンマ値を調整

### 4. プレビュー
「自動プレビュー」をオンにすると、パラメータ変更時に即座にプレビューが更新されます。

### 5. 保存
- **非破壊モード**: 「ソースを上書き」をオフにして、新しいファイルとして保存
- **上書きモード**: 「ソースを上書き」をオンにして、元のファイルを更新

## サンプルワークフロー

### 例1: 色調補正
1. テクスチャを読み込む
2. 色相シフトを30度に設定（暖色系に変更）
3. 彩度を1.2に設定（少し鮮やかに）
4. プレビューを確認
5. 新しいファイルとして保存

### 例2: HDR合成
1. ベーステクスチャを読み込む
2. 「テクスチャ合成」を展開
3. 合成テクスチャを指定
4. ブレンドモードを「HDRAdd」に設定
5. HDR カラーで発光色を指定（例: 青色で intensity 2.0）
6. 合成強度を調整
7. 保存

### 例3: マスク適用
1. テクスチャを読み込む
2. 「マスクを使用」をオンにする
3. マスクテクスチャ（グレースケール）を指定
4. 色調補正パラメータを調整
5. マスク強度で効果の強さを制御
6. 保存

### 例4: UVアイランドブラー
1. テクスチャとメッシュを読み込む
2. 「UVアイランドブラー」を展開
3. ブラー半径を5〜10に設定
4. ブラーシグマを調整（大きいほど滑らか）
5. プレビューでUVシームに滲みが出ないことを確認
6. 保存

## テストとデバッグ

### システム情報の確認
`Window > UniTex Editor > Check System Info` でCompute Shader対応状況を確認できます。

### 簡易テストの実行
`Window > UniTex Editor > Run Simple Test` で基本動作テストを実行できます。
テスト結果は `Assets/UniTexEditor_TestResult.png` に保存されます。

## トラブルシューティング

### Compute Shader が見つからないエラー
- `Resources` フォルダに `.compute` ファイルが配置されているか確認
- Unity エディタを再起動してみる

### プレビューが表示されない
- ソーステクスチャが正しく設定されているか確認
- Console ウィンドウでエラーメッセージを確認
- システム情報でCompute Shader対応を確認

### UV ブラーが動作しない
- メッシュが正しく指定されているか確認
- メッシュにUV座標が存在するか確認
- Console ウィンドウで「Found X UV islands」メッセージを確認

## パフォーマンスのヒント

- 4K以上の大きなテクスチャは処理に時間がかかる場合があります
- UVブラーは解析に時間がかかるため、最初は小さな半径で試してください
- 自動プレビューをオフにすると、パラメータ調整時の負荷を軽減できます
